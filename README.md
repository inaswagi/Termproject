# Termproject
The Personal Finance Tracker is a Python-based command-line application designed to help users manage and monitor their financial activity. The program enables users to record income and expenses, categorize transactions, view transaction history, and generate financial summaries. Additionally, it uses file handling techniques to store data persistently, allowing users to retain their financial records across multiple sessions.

The primary goal of this project is to provide a simple yet functional tool for personal financial management while demonstrating key programming concepts such as object-oriented programming, modular design, control structures, file handling, and error handling. The system is designed with usability in mind, offering a menu-driven interface that guides users through each feature in a clear and structured manner.
